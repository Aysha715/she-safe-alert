SheSafe is a lightweight single-page emergency web application designed to help women quickly send alerts, share live location, trigger fake calls, and access safety tools during emergency situations.

## Tech Stack
- HTML
- CSS
- JavaScript
- GitHub Pages

## Features
- One-tap emergency alert
- Send SMS with live Google Maps location
- Call emergency contact
- Fake call generator
- Loud alarm / siren
- Safety tips section

## Installation
No installation required.

1. Download or clone the repository
2. Open index.html in any browser

## Run Project
Open index.html using Chrome / Edge / Firefox.
